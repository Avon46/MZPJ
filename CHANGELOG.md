# 更新日誌

所有重要的專案變更都會記錄在此文件中。

## [1.0.0] - 2025-01-22

### 新增
- 🎉 專案初始版本
- 🏠 首頁、關於、菜單、最新消息、愛心公益頁面
- 📱 響應式設計支援桌機、平板、手機
- 🍽️ 菜單 SPA (單頁應用) 功能
- ❤️ 愛心公益數字 API 驅動系統
- 🔐 管理員介面用於更新愛心公益數字
- 📊 數字動畫效果和載入狀態
- 🔄 API 錯誤處理和重試機制
- 🚀 速率限制和基本安全措施

### 技術特色
- Flask 後端框架
- 模組化 JavaScript (ES6+)
- CSS 變數和 Grid/Flexbox 佈局
- SEO 優化和結構化資料
- 無障礙性 (ARIA) 支援
- 效能優化 (lazy loading, 節流/防抖)

### API
- `GET /api/love-stats` - 取得愛心公益統計數據
- `POST /api/love-stats` - 更新愛心公益統計數據 (管理員)

### 安全性
- CORS 設定
- 輸入驗證
- 速率限制 (每分鐘 100 次請求)
- XSS 防護

### 效能
- 圖片 lazy loading
- API 快取機制 (5分鐘)
- 前端快取管理
- 節流和防抖優化
